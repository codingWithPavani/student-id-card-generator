🪪 Student ID Card Generator

This beginner-friendly Python project inputs student details, generates a decorated ID card (text), and saves it as a `.txt` file.

🚀 Sample Output

╔══════════════════════════════════════╗
║         🪪 STUDENT ID CARD           ║
╠══════════════════════════════════════╣
║ Name     : Pavani Abothula           ║
║ College  : Sri Sivani College        ║
║ Branch   : CSE                       ║
║ Year     : 2                         ║
║ Roll No. : 12345                     ║
╚══════════════════════════════════════╝

📁 Files Included

  • student_id_card.py – main Python script
  • decorated_student_id_card.txt – saved output

🛠 How to Run

  1. Clone the repo
  2. Open it in an IDE (e.g., PyCharm)
  3. Run student_id_card.py
